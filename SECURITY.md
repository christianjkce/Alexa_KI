# Security Policy

## Reporting a Vulnerability
Bitte melde sicherheitsrelevante Probleme vertraulich an:

- E-Mail: info@jkce.de

Beschreibe:
- betroffene Komponente
- Reproduktionsschritte
- erwartetes vs. tatsaechliches Verhalten

Wir bestaetigen den Eingang so schnell wie moeglich und geben ein Update zum Status.
